# Messenger Back-End - FastAPI Practice Project (In Progress)

FastAPI framework practice project that implements:

* Scalable async FastAPI back-end messenger app.
* PostgreSQL DBMS with SQLAchemy ORM.
* Custom CRUD data controllers/repositories.
* Alembic PostgreSQL database migrations.
* Pydantic data validation and settings.
* PyTest unitests with fixtures and mocking (in progress).
* Docker container with version control (in progress).
